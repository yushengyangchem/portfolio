# Portfolio

Personal portfolio static site repository.

## Overview

- Main pages: `index.html`, `portfolio-zh.html`, `portfolio-en.html`
- Static assets: `assets/`
- Content source: `content/`
- Scripts: `scripts/`

## Local Development

This is a static project. You can preview it by opening `index.html` directly in your browser.

For local HTTP preview:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.
